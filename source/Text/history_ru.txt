История изменений:

2.8.1
- Для режима сравнения панелей оставлены только CRC32 и SHA1 (нет смысла использовать все алгоритмы).
- Мелкие изменения.

2.8.0
- Добавлена возможность проверки цифровой подписи для PE файлов.
- Макро-функция 'gethash' теперь возращает строку 'userabort' если пользователь прервал операцию.
- Исправлена проблема с подсчетом хеша через макрокоманду для символической ссылки на файл.
- Исправлена визуальная проблема с диалогом генератора хешей на новых версиях Far-а (спасибо ctapmex).

2.7.3
- Добавлена возможность сохранять хеши по одному файлу на каталог.
- Исправлена выдача из макроса хеша от предыдущего файла, если текущий файл невозможно прочитать.
- Добавлен новый опциональный параметр Quiet типа bool для макросов. Если выставлен в true, то сообщения об ошибках не будут показываться.
- Мелкие правки.

2.7.2
- Добавлена польская локализация (спасибо Andrzej Rudnik).
- Добавлена обрезка пробелов вокруг имени файла, переданного через префикс (ранее пробелы трактовались как часть имени).

2.7.1
- Исправлено отображение прошедшего времени в диалоге прогресса хеширования когда оно больше часа.
- Исправлена проблема с валидацией файлов доступных через Directory Junction.
- Обновлена библиотека RHash (должна увеличиться скорость некоторых алгоритмов).

2.7.0
- Добавлена возможность вызова плагина из lua макросов (см. IntChecker.GetFileHash.lua).
- Добавлена базовая функциональность для тестирования скорости хеширования.
- Добавлена опция игнорирования отсутствующих файлов при проверке.
- Исправлена невозможность загрузки хешлиста если пути были сохранены в формате UNC.
- Теперь SFV файлы могут иметь произвольное количество пробелов между путём файла и контрольной суммой.
- Обновлена испанская локализация (спасибо Mauro72).

2.6.2
- Исправлена неработающая опция сохранения хешей в верхнем регистре (регрессия).

2.6.1
- Исправлено падение при расчете скорости хеширования.

2.6.0
- Новый диалог прогресса хеширования.

2.5.1
- Исправлены проблемы с диалогом сравнения панелей (регрессия).

2.5.0
- Добавлена возможность использования алгоритма SHA3-512.
- Исправлено включение\выключение кнопки фильтра в диалоге генерирования хешей.
- Мелкие правки.

2.4.3
- Добавлена настройка вывода по умолчанию в диалоге генерирования хешей.
- Исправлена проблема с неправильным чтением некоторых настроек при старте.

2.4.2
- Исправлено падение при валидации файлов, пути которых содержат directory junction.
- Исправлен assert на некоторых именах с кириллицей.
- Добавлена возможность выбора кодовой страницы хеш-файла для операции проверки.

2.4.1
- Исправлена обработка Linux-style путей в хеш файлах.
- Исправлена проблема с порчей значения префикса в Far x64.
- Добавлена возможность выбора кодовой страницы файла при генерации хешей.

2.4.0
- Добавлена поддержка файловых фильтров при генерировании хешей (Far3 only).
- Мелкие правки.

2.3.5

2.3.0

- Мелкие правки.

2.2.1

- Исправлено падение при сравнении панелей с алгоритмами SHA-512 и Whirlpool.

2.2.0

- Исправлено неправильное определение алгоритма в некоторых случаях.
- Добавлена поддержка хеш листов в формате BSD.
